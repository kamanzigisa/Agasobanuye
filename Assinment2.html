1. What is CSS and its purpose?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. Its purpose is to control the layout, colors, fonts, and overall visual appearance of web pages.

2. What are the different ways to apply CSS to a webpage?

Inline CSS: Styles are applied directly within an HTML element using the style attribute.
Internal CSS: Styles are defined within a <style> tag in the <head> section of the HTML document.
External CSS: Styles are defined in a separate .css file linked to the HTML document using the <link> tag.

3. What is the box model in CSS?

The box model describes the rectangular boxes generated for elements in the document tree. It consists of:
Content: The actual content of the box (text, images).
Padding: Space between the content and the border.
Border: A line surrounding the padding (if any).
Margin: Space outside the border, separating the element from others.

4. Explain the difference between relative, absolute, fixed, and sticky positioning.

Relative: Positioned relative to its normal position. It moves without affecting the layout of other elements.
Absolute: Positioned relative to its nearest positioned ancestor (not static). It removes the element from the normal document flow.
Fixed: Positioned relative to the viewport, meaning it stays in the same place even when the page is scrolled.
Sticky: A hybrid of relative and fixed positioning. It behaves like relative until a certain scroll position is reached, then it becomes fixed.

5. What is specificity in CSS?

Specificity determines which CSS rule is applied when multiple rules could apply to the same element. It is calculated based on the types of selectors used: inline styles have the highest specificity, followed by IDs, classes, attributes, and finally, element selectors.

6. What are pseudo-classes and pseudo-elements?

Pseudo-classes: Keywords added to selectors that specify a special state of the selected elements (e.g., :hover, :focus).
Pseudo-elements: Allow styling of a specific part of an element (e.g., ::before, ::after, ::first-line).

7. What is the difference between display: none; and visibility: hidden;?

display: none; removes the element from the document flow, meaning it takes up no space.
visibility: hidden; hides the element but still takes up space in the layout.

8. What is a CSS preprocessor, and name a few examples?

A CSS preprocessor extends CSS with variables, nesting, and functions, allowing for more dynamic stylesheets. Examples include:
SASS (Syntactically Awesome Style Sheets)
LESS
Stylus

9. What is the z-index property and how does it work?

The z-index property controls the stacking order of overlapping elements. Higher values are stacked on top of lower values. It only works on positioned elements (those with position set to relative, absolute, fixed, or sticky).

10. Explain the concept of Flexbox.

Flexbox (Flexible Box Layout) is a CSS layout model that allows for the design of complex layouts more easily and efficiently. It enables responsive design by distributing space along a single axis (row or column) and aligning items within a container.

11. What are media queries in CSS?

Media queries allow CSS to be applied based on the conditions of the viewport, such as width, height, orientation, etc. They enable responsive design by applying different styles for different devices or screen sizes.

12. What is the difference between em, rem, and px in CSS?

px: A fixed unit representing pixels. It does not scale with user settings.
em: A relative unit based on the font size of the element. If the font size of the parent is changed, the size of the element using em will also change.
rem: A relative unit based on the root element's font size (usually <html>). It remains consistent regardless of parent element sizes.

13. Explain grid layout in CSS.

CSS Grid Layout is a two-dimensional layout system that allows for the creation of complex web layouts using rows and columns. It provides control over the placement of items in a grid format, enabling responsive designs with ease.

14. What is the difference between inline, block, and inline-block elements?

Inline: Elements that do not start on a new line and only take up as much width as necessary (e.g., <span>, <a>).
Block: Elements that start on a new line and take up the full width available (e.g., <div>, <h1>).
Inline-block: Elements that are formatted as inline elements but can have width and height set (e.g., <img>).

15. What is the purpose of calc() in CSS?

The calc() function allows for dynamic calculations in CSS property values. It can combine different units (e.g., calc(100% - 50px)) for responsive layouts.

16. Explain the opacity and rgba() in CSS.

opacity: A property that sets the transparency level of an element, where 0 is fully transparent and 1 is fully opaque.
rgba(): A color function that defines colors using red, green, blue, and an alpha value for transparency (e.g., rgba(255, 0, 0, 0.5) for semi-transparent red).

17. What is the difference between min-width and max-width?

min-width: Sets the minimum width of an element. The element cannot be smaller than this value.
max-width: Sets the maximum width of an element. The element cannot exceed this value.

18. What is the difference between transitions and animations in CSS?

Transitions: Allow for smooth changes between two states of an element (e.g., changing color on hover). They are triggered by state changes (like hover).
Animations: More complex sequences that can run independently of state changes. They can involve multiple keyframes and can repeat or run automatically.

19. How do you center an element horizontally and vertically using CSS?

Horizontal centering: Use margin: auto; on a block element with a set width, or use text-align: center; for inline elements.
Vertical centering: Use Flexbox by setting display: flex; and align-items: center; on the parent, or use grid layout with align-content: center;.

20. What is the purpose of box-sizing: border-box;?

The box-sizing: border-box; property changes the box model so that padding and borders are included in the element's total width and height. This makes it easier to size elements without worrying about additional padding or borders affecting the overall size.
