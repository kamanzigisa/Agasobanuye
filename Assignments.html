Individual Assignment 1

1. What does HTML stand for?

HTML stands for <HyperText Markup Language>.

2. What is the purpose of HTML?

HTML is used to structure content on the web.

3. What is the basic structure of an HTML document?

The basic structure includes <!DOCTYPE html>
<html>
<head> and
<body> tags.

4. What are HTML elements?

HTML elements are the building blocks of HTML, defined by tags.

5. What is a tag in HTML?

A tag is a keyword enclosed in angle brackets
Example: <tagname>

6. What is an attribute in HTML?

Attributes provide additional information about an element, defined in the opening tag.

7. What is the difference between block-level and inline elements?

Block-level elements take up the full width,
while inline elements take up only as much width as necessary.

8. What is the purpose of the head tag?

The <head> tag contains meta-information about the document,
like the title and links to stylesheets.

9. What is the title tag used for?

The <title>tag specifies the title of the web page,
shown in the browser's title bar.

10. How do you create a hyperlink in HTML?

A hyperlink is created using the <a>tag.
Example: <a href="url">Link Text</a>

11. What is an image tag in HTML?

The <img> tag is used to embed images. Example: <img src="image.jpg" alt="description">.

12. What is the purpose of the alt attribute in images?

The alt attribute provides alternative text for an image if it cannot be displayed.

13. How do you create a list in HTML?

Lists can be created using <ul> for unordered lists and <ol> for ordered lists.
Example:
<ul>
<li>Item</li>
</ul>.

14. What is the div element used for?

The <div> element is a container for other elements, commonly used for layout purposes.

15. What is the purpose of the span tag?

The <span> tag is used to group inline elements for styling purposes.

16. How do you create a form in HTML?

A form can be created using the <form> tag, with various input elements.
Example:
<form>
<input type="text">
</form>.

17. What is the input element?

The <input> element is used to create interactive controls for forms.

18. How do you create a dropdown list in HTML?

A dropdown list can be created using the <select> tag.
Example: <select><option>Choice</option></select>.

19. What are semantic HTML elements?

Semantic elements clearly describe their meaning in a human- and machine-readable way,
Example<header>, <footer>.

20. What is the purpose of the header element?

The <header> element is used to contain introductory content or navigational links.

21. How do you include CSS in an HTML document?

CSS can be included using the <link> tag in the <head> section or using the <style> tag.

22. What is the script tag used for?

The <script> tag is used to embed JavaScript code within an HTML document.

23. What is the purpose of the meta tag?

The <meta> tag provides metadata about the HTML document, such as character set and description.

24. How do you create a table in HTML?

A table can be created using the <table> tag, with
<tr> for rows and
<td> for cells.
Example:
<table>
<tr>
<td>Data</td>
</tr><
/table>.

25. What is the th element used for?

The <th> element is used to define header cells in a table.

26. What is the purpose of the colspan attribute?

The colspan attribute specifies the number of columns a cell should span in a table.

27. How do you create a responsive web design?

Responsive web design can be achieved using CSS media queries to adapt layouts to different screen sizes.

28. What is the role of the viewport meta tag?

The viewport meta tag controls the layout on mobile browsers.
Example: <meta name="viewport" content="width=device-width, initial-scale=1">.

29. How do you include external JavaScript in HTML?

External JavaScript can be included using the <script> tag with the src attribute.
Example: <script src="script.js"></script>.

30. What is the purpose of the iframe tag?

The <iframe> tag is used to embed another document within the current HTML document.

31. How do you create a comment in HTML?

Comments are created using <!-- comment text -->.

32. What is the purpose of the canvas element?

The <canvas> element is used for drawing graphics via scripting (usually JavaScript).

33. How do you create a hyperlink that opens in a new tab?

A hyperlink can open in a new tab using the target attribute.
Example: <a href="url" target="_blank">Link Text</a>.

34. What is the blockquote tag used for?

The <blockquote> tag is used to indicate that the enclosed text is a quotation from another source.
